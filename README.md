# EcoSort
EcoSort AI is an AI-powered sustainability assistant built using IBM Granite models through IBM watsonx.ai. The application helps users identify waste categories and provides proper disposal, recycling, and sustainability recommendations.

The project aims to improve waste segregation awareness and encourage responsible waste management practices to support environmental sustainability.

Problem Statement

Improper waste segregation is a major environmental challenge. Many recyclable, hazardous, and electronic waste items are discarded incorrectly due to lack of awareness. This leads to increased pollution, landfill waste, and loss of valuable recyclable resources.

EcoSort AI addresses this issue by providing intelligent guidance on waste classification and disposal methods using Artificial Intelligence.

How It Works

1.User enters a waste item.
2.IBM Granite analyzes the input.
3.The system classifies the waste.
4.Disposal and recycling recommendations are generated.
5.Sustainability tips are provided to the user.
